# Instruções de execução do projeto

## Clonar projeto

```
git clone https://github.com/slthiago/TesteAlpesOne.git
```

## Acessar projeto

```
cd TesteAlpesOne
```

## Instalar dependências

```
npm install
```

### Compilar projeto

```
npm run serve
```

### Compilar projeto para produção

```
npm run build
```
